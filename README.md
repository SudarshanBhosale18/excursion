# Excursion

Excursion is a simple and elegant webpage project built using HTML and CSS. It is based on a design and redline mock that guides its layout, spacing, and styling. This project is ideal for learning and practicing basic web development and Git/GitHub workflows.

## 📁 Project Structure

excursion/
├── index.html
└── resources/
├── css/
│ └── style.css
├── images/
│ └── [image files]
└── videos/
└── [video files]


## 🚀 Features

- Responsive HTML/CSS layout
- Embedded video
- Custom styling with external CSS
- Git version control and GitHub hosting
- Organized project file structure
